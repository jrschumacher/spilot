spilot
======

ServerPilot CLI

```
  Usage: spilot [options] [command]

  Commands:

    auth:save <client_id> <client_key>  Save authentication
    server
    server:list

  Options:

    -h, --help       output usage information
    -V, --version    output the version number
    -i, --id <id>    Client id [required]
    -k, --key <key>  Client key [required]
```
